NAME

mimesender - an example of integrating MIME-tools with Mail::Send.

SYNOPSIS

Usage: 

    mimesender YOU@YOUR.HOST

DESCRIPTION

This just fakes up a little message and sends it by using Mail::Send and the 'sendmail' handler.
