The GD.Graph.mixed plugin provides an interface to the GD::Graph::mixed class defined by the GD::Graph module. It allows one or more (x,y) data sets to be plotted with various styles (lines, points, bars, areas etc).

See [% ttlink('GD::Graph') -%] for more details.
