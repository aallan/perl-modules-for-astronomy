=head1 NAME

XML::LibXML::CDATASection - The DOM CDATASection Class

=head1 synopsis

 use XML::LibXML

 $node = XML::LibXML::Comment( $content );


=head1 DESCRIPTION




This class provides all functions of
B<XML::LibXML::Text>, but for CDATA nodes.




=head2 Methods

=over 4

=item B<new>




The constructor is the only provided function for
this package. It is required, because
B<libxml2> treats the different textnode
types slightly different.






=back
